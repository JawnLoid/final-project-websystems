Featured Projects

01.
JavaRise

JavaRise is a 2D pixel-art game that bridges the gap between playing and programming by embedding real programming mechanics directly into the core gameplay loop. Instead of separating coding from the gaming experience, JavaRise treats programming itself as a game mechanic — players must think and act like developers to progress through the game world.

The game is built entirely in 2D pixel art, with every sprite crafted from scratch using Aseprite. Each character, environment tile, and animation is an original asset designed specifically for JavaRise, giving the game a distinct handcrafted visual identity.

Currently in active development as part of my IT studies at STI Tanay, with more features, levels, and screenshots coming soon.
